英語の解説



get close"get A"は「Aを手に入れる」という意味だけではなく「Aになる」という意味もあります。
（例）"It's getting cold. I think we should go inside."「だんだん寒くなってきたな。僕たち家の中に行くべきだと思うよ。」

make it to the end of December"Make it to A"で「Aまでたどり着く」「Aまで耐え抜く」という意味です。
（例）"We made it to Disney land!"「ディズニーランドにたどり着いた！」

ways
way"way"は「道」という意味だけではなく「方法」「やり方」という意味もあります。
（例）"Try to do it in a different way."「他の方法を試してみなよ。」

Looking for the path
looking for a way out"Look for A"で「Aを探す」「Aを探し求める」という意味です。
（例）"I've been looking for my phone for about 2 hours."「私の携帯を約２時間探しています。」

like"like"は「好き」という意味だけではなく「〜のような」「〜と同じ」という意味があります。
（例）"It sounded like my dad."「今のお父さんのような声だったな。」

good enough for anything"Enough for A"で「Aをするのに十分」「Aのために十分」という意味です。

run away"Run away"は「逃げる」「逃亡する」「家出する」という意味です。

Wide-eyed"Wide-eyed"は「目を見開く」「目を丸くする」という意味です

dove"pigeon"と同じ「ハト」と訳されるが"dove"は「白く小さめの、白い幸せの象徴のハト」のことを言います。また「純粋な」「優しい」と訳されることもあります。

set you free"Set A free"は「Aを自由にする」という意味です。
（例）"I will set you free, Genie!"「君を自由にするよ、ジーニー！」


Zero （ゼロ ）についての解説


この曲は映画の主題歌として有名なアメリカのバンドImagin Dragons（イマジン・ドラゴンズ）によって歌われています。
シンデレラは、彼女の映画では耳が描かれていないませんが、この映画では耳が描かれています。
